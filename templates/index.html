Technology used

HTML

HTML is the standard markup language used to create web pages. It defines the structure of a webpage using tags and elements such as headings, paragraphs, links, images, and more. HTML is essential for web development, providing the basic foundation for designing and displaying content on the internet...

Javascript

JavaScript is a scripting language used to create dynamic content on web pages, such as interactive forms, animations, and real-time updates. It runs in the browser and works with HTML and CSS to enhance user experience and web functionality.

Machine Learning

The model detects, if a text message/mail is spam(1) or ham(0) using techniques like tokenizaton, multinomial naive bayes classifier, etc.

CSS

CSS (Cascading Style Sheets) is used to style and design HTML elements on web pages. It controls layout, colors, fonts, spacing, and responsiveness, enhancing visual presentation and user experience.

Flask

Flask is a lightweight Python web framework used to build web applications. It handles routing, templates, and request processing, making it ideal for connecting front-end forms to machine learning models.

Dataset

The dataset used is a .csv file containing labeled text messages as spam or ham. It is used to train and test the model with features extracted for accurate classification.
